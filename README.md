## vue3 开发环境搭建
```sh
# 系统内如果安装了旧版本vue-cli需要先删除
npm uninstall vue-cli -g
# 安装 vue3  (node版本8.11.0+)
npm install -g @vue/cli
#  创建项目
vue create iskyin-fe
# 安装依赖
npm install
# 用于开发的编译和热加载环境
npm run serve
# 用于生产环境的编译和压缩
npm run build
# 跑测试
npm run test
# 自定义配置链接

```
